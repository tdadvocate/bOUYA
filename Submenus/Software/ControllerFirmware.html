Controller Firmwares

*This page is still under construction and will be updated in a future alpha build with more information.*
Below you can find a quick list of firmware versions for the controllers and some information about them when available. More research needs to be done yet on these firmwares as so far I have not found too much information about them or how to upgrade/downgrade them yet. If you have any information on how to do this, please let us know so that we may make a page with how to do so. You can check the firmware of your controller by going to the controller pairing screen. Once here you should be able to press the "Y" button on the controller then use your touchpad to click on the settings cog icon in the top right of the screen. This should pull up a screen that shows the battery level of the controller as well as its firmware version. Please let us know if you have a firmware that is not listed below and any information that you can regarding it.

0x0004

Not much information has been discovered about this firmware version yet. It is possible that this was a typo or false information but I have been unable to determine a definitive answer.

0x0101

These appear to have been shipped with the developer systems.

0x0102

Likely shipped with most early backer and kickstarter consoles and separate controllers.

0x0103

It appears that this was in some launch day system packaging.

0x0104

This appears to be an updated firmware via RMA and any systems starting with the Anthracite (Black) systems.

0x0104 1.4

Not sure if this is different from 0x0104 or if they are the same version.
